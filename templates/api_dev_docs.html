ppdpAPIService

ppdpControllers.controller('develop', ['ppdpAPIService',
function(ppdpAPIService) {
...
}]
);

ppdpAPIService is a custom angulajs api which allows angularjs to communicate with the ppdp .NET api

ppdpAPIService must be injected like code above

.account

Object

.login

Function
ppdpAPIService.create(params);

create new account object


.logout

Function
ppdpAPIService.logout();

retrieves list of account objects


.isloggedin

Function
ppdpAPIService.isloggedin();

saves account object
