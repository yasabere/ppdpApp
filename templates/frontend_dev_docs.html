Options
Name type default description
ng-Model * variable N/A AngularJs model variable which is bound to input.
options * array [objects] null Array of objects to be displayed as dropdown
text-Field string 'text' String name of object field to be shown as dropdown option label.
value-Field string 'text' String name of object field which contains value of selected option.
description-Field string 'text' String name of object field which contains description to be shown beside dropdown option label for each option.
placeholder string 'text' Place holder text which appears when nothing is selected.
extraOptions array 'text' Options which apear at bottom of dropdown.
returnObject boolean false Shows the modal when initialized.
search boolean false Shows the modal when initialized.
disabled boolean false Shows the modal when initialized.
